h1. Ruby I18n-inflector

Simple inflection support for Ruby's I18n.

h2. Installation

gem install i18n-inflector

h2. Tests

You can run tests both with

* `rake test` or just `rake`
* run a test file directly, e.g. `ruby -Ilib -Itest test/inflector_test.rb`

h2. Usage

See "module documentation":http://rubydoc.info/gems/i18n-inflector/ for more info about usage.
